# Customer Segmentation Project

## Overview
This project uses K-Means Clustering to segment customers based on income and spending score.

## Tools Used
- Python
- Google Colab
- Pandas
- Scikit-learn
- Power BI

## Features
- Data Cleaning
- EDA
- K-Means Clustering
- Power BI Dashboard

## Dataset
Kaggle Mall Customers Dataset
